moveToLobbyHandler(data)

The event handler for the moveToLobby signal from the server. It determines if the request to join or create a lobby was successful or not. If not, it creates an alert with the relevant error message and returns user to the join view. If the server signals success, the Game object is instantiated. The lobbyCode is set in localStorage as sent by the server, the lobbyMenu view is setup and made visible, and a request for the player list is sent.
